4 Task 4

4.1 Create Height v BHD Plot with new Quad Curve

quad.lm=lm(Height~BHDiameter + I(BHDiameter^2),data=spruce.df)
myplot=function(x){
  quad.lm$coef[1] +quad.lm$coef[2]*x  + quad.lm$coef[3]*x^2
}
plot(Height~BHDiameter, data = spruce.df)
curve(myplot, lwd=2, col="steelblue",add=TRUE)

4.2 Storing Res and Fit Variables

quad.fit <- fitted(quad.lm)
quad.res <- residuals(quad.lm)

4.3 Plot with Quadratic Residuals v Fitted Values

plot(quad.res ~ quad.fit)

4.4 Quadratic Norm Check and Shapiro

normcheck(quad.lm,shapiro.wilk = TRUE)
